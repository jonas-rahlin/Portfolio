The Forty Servants App

The Forty Servants App is a divination card application based upon Tommie Kellys divination/oracle deck. It uses JavaScript to pull random cards from the deck and then displays them on the screen. It also displays various info attached to each card.

It is made with the permission of Tommie Kelly and you can find out more about him on his site adventuresinwoowoo.com.

Badass Burgers

Badass Burgers is a restaurant page for a fictional hamburger restaurant. It uses JavaScript for opening and closing modals, disabling the main window and to create certain navlink effects, such as sticky text-decorations.
